# 🏠 Housing Price Prediction using Linear Regression

A Machine Learning project focused on predicting housing prices using structured data. This project implements and evaluates a Linear Regression model to understand how different features impact property valuation.

---

### 🛠️ Tech Stack & Libraries
- **Language:** Python 🐍
- **Libraries:** NumPy, Pandas, Scikit-Learn, Matplotlib, Seaborn

---

### 📊 Dataset & Features
The model is trained on real estate data including key features such as:
- **Area:** Total square feet of the property
- **Bedrooms/Bathrooms:** Number of rooms
- **Location:** Neighborhood quality index

---

### 🚀 Key Learning Outcomes & Implementation
- **Data Preprocessing:** Handled missing values, outliers, and data cleaning using **Pandas**.
- **Feature Scaling:** Applied normalization to improve model convergence.
- **Model Training:** Implemented and trained the **Linear Regression** model using **Scikit-Learn**.
- **Evaluation:** Evaluated model performance using **Mean Squared Error (MSE)** and **R-squared ($R^2$) Score**.
